# Pennywise-CS253 Project

Pennywise is a personal finance management system designed to help users track income, expenses, budgets, and savings efficiently.  
This project is being developed as part of academic coursework.

## Project Status
Ongoing

## Features
- User registration and secure authentication
- Income and expense tracking with category management
- Monthly and category-wise budget planning
- Budget utilization tracking with alerts
- Financial analytics and reporting

## Project Structure
- `user/` – User authentication and profile management  
- `income/` – Income management module  
- `expense/` – Expense tracking and categorization  
- `budget/` – Budget creation and utilization tracking  
- `category/` – Default and custom category management  
- `analytics/` – Reports and financial analysis  
- `docs/` – SRS and design documents  

## Technologies Used

- **Programming Language:** JavaScript  
- **Frontend:** React, HTML, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Tools:** GitHub, UML Diagrams, Visual Studio Code, Postman  
- **Protocols:** HTTP/HTTPS
